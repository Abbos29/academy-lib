Условный оператор if

Оператор if

Используя условный оператор if (от англ. "если"), можно выборочно выполнять отдельные части программы в зависимости от какого-то условия. Например, мы знаем возраст пользователя нашего приложения. Если пользователю есть 18 лет, мы позволяем ему войти в приложение. Если же ему менее 18 лет, доступ будет запрещен.

Ниже приведена простейшая форма оператора if: 

if (условие) {
  код, который надо выполнить, если условие соблюдается
}

В условии записывается какое-либо логическое выражение. Логическое выражение можно представить себе как вопрос, на который можно ответить только "Да" или "Нет".

Например, запись 10 < 11 означает вопрос - 10 меньше чем 11? Ответ на этот вопрос - "Да". Если то же самое записать в виде кода, выглядеть это будет следующим образом:

if (10 < 11) {
  console.log("10 меньше 11");
}

В данном случае программа оценивает условие, и если оно истинно, код внутри условия выполняется, то есть в консоль будет выведена строка текста 10 меньше 11. Если же условие ложно - код внутри условия не выполняется.

В примере выше числовое значение 10 меньше чем 11, и поэтому условное выражение истинно, в результате чего выполняется код в фигурных скобках. Еще один пример с противоположным условием:

if (10 < 9) {
  console.log("Этa строка не отобразится");
}

В данном случае условие 10 меньше 9 не выполняется (ответом на вопрос 10 меньше чем 9? будет "Нет"), поэтому метод console.log() не будет вызван.

Базовый набор операций сравнения, которые можно использовать в логических условиях, перечислен ниже:

Операция Значение
< Меньше
<= Меньше или равно
> Больше
>= Больше или равно
== Равно (проверяет равно ли одно значение другому, не путайте с оператором присваивания "=")
=== Строгое равно - проверяет равно ли одно значение другому без приведения типа
!= Не равно
!== Не равно (без приведения типа)

Логично будет спросить - зачем писать такой код, как в примере выше, если заранее известно, что 10 больше чем 9, и строка кода никогда не будет выполнена? Смысл в том, что все те же самые операции можно проводить с переменными. Вспомните самый первый пример в данной статье. Например, у нас есть переменная, в которой хранится возраст пользователя, и у каждого пользователя он будет разный. Если пользователю есть 18 лет, мы выполним один кусок кода. Если же пользователю менее 18 лет, выполнится другой кусок кода, который запрещает пользователю доступ в приложение.

Условие else

Очень часто при оценке условия с помощью оператора if нам надо также выполнить какое-то действие, если условие не выполняется. Для этого используется ключевое слово else (от англ. "иначе"). Пример с проверкой возраста пользователя выше можно переписать следующим образом:

if (userAge >= 18) {
  код, который пускает пользователя на сайт
} else {
  код, который не пускает пользователя на сайт
}

Несколько условий, else-if

Мы также можем проверить несколько условий, выполнив в итоге только первое из них, которое будет истинным. Например, мы просим пользователя заполнить его возраст на сайте, и в зависимости от введенного значения, будет показывать разное сообщение. Выполнить даное действие можно с помощью команды else if:

if (userAge < 18) {
  alert("А родители разрешили?");  
} else if (userAge < 100) {
  alert("Добро пожаловать!");
} else {
  alert("А вам точно больше 100 лет???");
}

В итоге каждое из этих условий будет по очереди проверяться, пока не будет найдено истинное условие. Все остальные условия не будут выполнены. Проще говоря, в конструкции if - else if - else всегда может быть выполнено только одно условие.

Тернарный оператор ?

В Javascript существует также более короткая запись оператора if - это так называемый тернарный оператор ?, который позволяет в одной строке записать оба действия - как при выполнении условия, так и при его невыполнении.

условие ? код если условие истинно : код если условие ложно

Тернарный оператор можно использовать только в том случае, если код, который необходимо выполнить в одном и другом случае, можно записать в одну строку. Например:

userAge < 18 ? alert("А родители разрешили?") : alert("Добро пожаловать!")

Обратите внимание, что код при выполнении и невыполнении условия разделен двоеточием.

Задача

Код Math.floor(Math.random() * 6) + 1 позволяет загадать случайное число от 1 до 6. Напишите программу, которая загадывает такое случайное число, а потом просит пользователя его угадать (С помощью prompt). Если пользователь угадал число, необходимо вывести сообщение Поздравляем!. Если загаданное число меньше того, что ввел пользователь, необходимо вывести сообщение Меньше!. Если наоборот, вывести сообщение Больше!

Нажмите, чтобы подсмотреть решение 

const number = Math.floor(Math.random() * 6) + 1

const userInput = +prompt("Загадайте число от 1 до 6")

if (number < userInput) {
  alert("Меньше!")
} else if (number === userInput) {
  alert("Поздравляем!")
} else {
  alert("Больше!")
}
