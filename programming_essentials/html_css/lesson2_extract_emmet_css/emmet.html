Emmet

EMMET

Что такое EMMET?

EMMET - это плагин, позволяющий ускорить верстку, используя сокращенные команды и их комбинации. Он устанавливается и работает в таких редакторах, как Sublime Text, Atom, Brackets, WebStorm, Notepad++, CodePen и многих других.

Для его использования в коде достаточно ввести сокращенную команду и нажать Tab.

Например:

HTML

ul>li*3

Результат:

<ul>
  <li></li>
  <li></li>
  <li></li>
</ul>
CSS

fz18+fw700

font-size: 18px;
font-weight: 700;

Как подключить плагин?

Brackets:

В верхнем меню выберите Файл > Менеджер расширений > Доступные, введите в поиске Emmet и нажмите Установить. Плагин готов к использованию.

IntelliJ IDEA или Webstorm:

В верхнем меню выберите File > Settings > Editor > Emmet. Там находятся все его настройки. Скорее всего, после установки программы плагин уже будет включен по умолчанию.

Atom:

В верхнем меню выберите Settings > Install, в поиске введите Emmet и нажмите Install.

Как можно заметить, схема установки плагина в различных IDE примерно одинакова.

Emmet в HTML

Теги-сокращения пишутся без скобок

p

<p></p>

Класс тега пишется после точки .

div.class1.class2

<div class="class1 class2"></div>

ID тега пишется после символа #

div#header

<div id="header"></div>

Атрибуты пишутся внутри квадратных скобок [] через пробел

td[rowspan=3 colspan=5]

<td rowspan="3" colspan="5"></td>

Текст идет внутри фигурных скобок {}

h1{Title}

<h1>Title</h1>

Соединяются соседние теги через +

h1{Title}+h2{text}

<h1>Title</h1>
<h2>text</h2>

После > пишутся дочерние элементы

ul>li>a

<ul>
  <li>
    <a href=""></a>
  </li>
</ul>

Число тегов, которые надо продублировать, указывается после *

ul>li*3

<ul>
  <li></li>
  <li></li>
  <li></li>
</ul>

Несколько тегов группируются в скобках ()

ul>(li>a)*2

<ul>
  <li>
    <a href=""></a>
  </li>
  <li>
    <a href=""></a>
  </li>
</ul>

Подняться на уровень выше можно через ^

div>h1^div>p

<div>
  <h1></h1>
</div>
<div>
  <p></p>
</div>

Нумерация проставляется автоматически знаком $, начиная с 1

ul.list>li.item$*3

<ul class="list">
  <li class="item1"></li>
  <li class="item2"></li>
  <li class="item3"></li>
</ul>

Нумерация с конкретного числа пишется после $@

ul.list>li.item$@5*3

<ul class="list">
  <li class="item5"></li>
  <li class="item6"></li>
  <li class="item7"></li>
</ul>

! создает стартовый скелет HTML5 страницы:

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8" />
<title>Document</title>
</head>
<body>

</body>
</html>

Слово lorem добавляет стандартный текст, количество слов указывается числом в конце. Если число не указано, текст добавится в полном размере.

h1>lorem8

<h1>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</h1>

  <img src="" alt="img" class="logo-img">

Emmet в CSS

В CSS используются два символа: + - для соединения сокращенных свойств в комбинацию; - - используется для описания нескольких значений сразу.

Например:

bd2-s-red+oln

border: 2px solid red;
outline: none;

Существует огромное количество сокращений в СSS. Многие из них используются крайне редко, ниже приведен список часто употребляемых.
